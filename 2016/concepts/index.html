Anchor text

The anchor text is the clickable text of a link. The anchor text of a link embedded within an image is normally defined by the hidden "AltText" tag inserted into the web page.



Link

A link is a clickable URL on a web page. Most will appear in plain text, but some will be embedded inside an image that you can click on. Click on this link or this arrow to go to the next section

Majestic terms

Backlink

A backlink is an incoming link from another site. For example, if site A had a link to site B on one of its web pages, then site B has a backlink to site A.

Root domain

A root domain is synonymous with domain, and includes the domain name and top-level domain.


http://www.example.com/blog

Referring domain

A domain refers to another domain if it has at least one link to it on any of its web pages.

Trust Flow

Trust Flow is a Majestic trademarked term that represents the quality of a site. Sites that are linked with a site with high Trust Flow will themselves have their trust flow increased, and the opposite if they are linked with a site with low trust flow.


Trust flow was originally allocated based on a manual review of the web's highest quality sites.

Citation Flow

Citation Flow is another Majestic trademarked term that represents the "power" that the link or site carries, based on the number of backlinks it has. A high Citation Flow can indicate that a site is rich in web 'spam' or that it is just very popular.


The Majestic Flow Metrics (Trust Flow and Citation Flow) go hand in hand when determining a site's quality.

Fresh and Historic indexes

The Historic Index is the largest index of backlink history on the web and spans back more than 5 years. As building it takes days of processing power, the Fresh Index was introduced as an up-to-date database that contains crawled data from the past 90 days and is updated throughout the day.
